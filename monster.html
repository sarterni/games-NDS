Bonjour à tous Bienvenue dans monster Game le but du jeu est très simple. Assurer la survie du monstre à l'aide des différends bouton d'action verts. Amusez-vous bien ! Nouveau votre jouer jouable avec les touches de clavier :
H : help
R : Run
F : fight
W : work
E : eat
S : sleep
N : new life
K : Kill
D : Activer désactiver le Dark mode
Pour utiliser les touches de clavier pensez à vous mettre en minuscule.
